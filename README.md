# guestbook

MERN Stack Application

# To run app

### 1-backend

```
cd backend
source exports.sh  "linux"

or
for linux
 export CP_AUTH_SECRET=anykey
 export DB="mongodb://localhost/guestbook"

 for windows
 set CP_AUTH_SECRET=anykey
 set DB="mongodb://localhost/guestbook"

npm start
```

### 2-FrontEnd

```
cd frontend
npm i
npm start
```
